# Tasks Convention

`/tasks` refers to the living task backlog for this repository.

## Canonical File

The source of truth is:

- [docs/tasks.md](/Users/theivendrankabelan/Documents/python_scripts/incident_triaging_n_resolutions/docs/tasks.md)

## How To Use It

When you want to update the implementation backlog, use `/tasks` in your request and describe the change.

Examples:

- `/tasks add a task for Azure Blob raw storage`
- `/tasks simplify Phase 2 to only CSV ingestion and KB retrieval`
- `/tasks reorder Phase 1 so MCP server comes before FastAPI`

## Editing Rule

- Update `docs/tasks.md` directly.
- Keep tasks small, testable, and stepwise.
